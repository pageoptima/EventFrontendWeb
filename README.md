# Events

Events is a social events platform where people can share upcoming events, post reels and photos, and discover what is happening in their community.

## Features

- Create and share upcoming events
- Post reels and photos
- Feed-style discovery for new content
- Fast, responsive UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- ESLint

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Preview the production build

```bash
npm run preview
```

## Scripts

- `npm run dev` - Start the dev server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint
